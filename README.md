# Azure Identity Architecture Lab

## Purpose

This repository contains hands-on labs, architecture notes, automation scripts, and troubleshooting guides for mastering Azure Authentication and Authorization.

## Learning Framework

TRAP

- Test
- Retain
- Associate
- Perform

## Topics

- JWT
- OAuth
- Client Credentials
- Auth Code + PKCE
- Entra ID
- Managed Identity
- API Scopes
- RBAC vs Claims
- Multi-Tenant Architecture
- Conditional Access

## Goal

Become capable of:

- Designing enterprise authentication systems
- Troubleshooting identity issues
- Implementing secure integrations
- Teaching authentication concepts
- Performing Solution Architect responsibilities
